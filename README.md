# Controle de Refeição App

## Introduction
Controle de Refeição is a Ruby on Rails application designed for users to manage their meals,
track water intake, count steps, and schedule appointments with nutritionists.

This app helps maintain a healthy lifestyle by providing insights and easy management
of health-related activities.

## Features
- User authentication system.
- Ability to track daily water consumption.
- Step counting functionality.
- Scheduling and managing appointments with nutritionists.
- Meal logging with detailed descriptions and calorie counts.

## Tech Stack
- **Framework**: Ruby on Rails
- **Database**: PostgreSQL
- **Frontend**: HTML, CSS, JavaScript
- **Runtime**: Ruby 3.2.2

## System Dependencies
- Ruby: 3.2.2
- PostgreSQL: v15.4
- Rails: 7.0.7
- Node.js: v16.15.1
- Yarn: 1.22.19

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

Ensure you have the following installed on your local machine:

- Ruby 3.2.2
- PostgreSQL v15.4
- Node.js v16.15.1
- Yarn 1.22.19

### Installation

1. Clone the repository:

```bash
git clone https://github.com/SeixasCco/controle_refeicao.git
cd controle_refeicao
```

2. Install Ruby dependencies:

```bash
bundle install
```

3. Install JavaScript packages:

```bash
yarn install
```

4. Setup the database:

```bash
rails db:create db:migrate db:seed
```

5. Start the server:

```bash
rails server
```

## Usage

After starting the server, you can use the application in your browser:

`http://localhost:3000/`

## Testing

To run the tests, you can use the following command:

```bash

```
